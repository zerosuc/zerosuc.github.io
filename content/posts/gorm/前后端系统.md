## 说明

#### 技术栈

- 语言：Golang、Vue+Typescript；
- 后端：gin gorm 、Casbin；
- 前端：Vue3、Vite、Element-Plus、TypeScript、Pinia 等；

### 后端项目运行

- git clone
- 起来 mysql 和 redis
- go run main.go

### 前端运行

- npm i
- npm run dev
- npm run build:prod 打包
